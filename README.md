# OpenGL Subway Surfers Clone

A Subway Surfers–inspired game built with a custom engine using OpenGL.

<img width="2525" height="1537" alt="image" src="https://github.com/user-attachments/assets/a2b972bf-9d31-4689-a931-87f0e1492119" />

# Features
 - OpenGL rendering with basic lighting and animation
 - Catmull–Rom spline–based track system
 - Custom Entity Component System (ECS)
 - Custom event system
 - Custom collision detection system
